# Auto bt4g - Android App

## فتح المشروع في Android Studio

1. افتح Android Studio
2. File → Open → اختار فولدر AutoBt4g
3. استنى الـ Gradle يخلص Sync
4. Build → Build APK(s)
5. هتلاقي الـ APK في: app/build/outputs/apk/debug/

## طريقة الاستخدام

- افتح التطبيق واكتب اسم الفيلم أو المسلسل
- أو: من أي تطبيق تاني (Telegram مثلاً) → Share → Auto bt4g
- التطبيق هيدور على bt4g ويجيب رابط الماجنت أوتوماتيك
- بعدين هيفتح تطبيق التورنت اللي عندك (LibreTorrent / µTorrent)

## ملاحظة
